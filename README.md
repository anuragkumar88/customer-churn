## 📌 Overview
This project uses a Machine Learning model to predict whether a customer will churn or not.  
It helps businesses make better decisions and reduce customer loss.

---

## 🎯 Problem Statement
Customer churn is a major challenge for companies.  
This project analyzes customer data and predicts churn using ML techniques.

---

## 🛠️ Tech Stack
- 🐍 Python  
- 📊 Pandas, NumPy  
- 🤖 Scikit-learn  
- 📈 Matplotlib  

---

## 🧠 Model Used
- 🌳 Random Forest Classifier

## ⚙️ How to Run

### 🔹 1. Clone Repository

git clone https://github.com/anuragkumar88/customer-churn.git

cd customer-churn


### 🔹 2. Install Dependencies

pip install -r requirements.txt


### 🔹 3. Run Main App

python app.py


### 🔹 4. Run Dashboard 📊

python src/dashboard.py


### 🔹 5. Run Full Pipeline ⚙️

python main.py
